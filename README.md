<h2>Installation</h2>

<ol>
  <li><b>Extract:</b> Download the files from the link above and extract the folder to your computer.</li>
  <li><b>Settings:</b> Open <b>Kaset</b>, go to the <b>toolbar</b>, and open <b>Settings</b>.</li>
  <li><b>Extensions:</b> Navigate to the <b>Extensions</b> tab.</li>
  <li><b>Add:</b> Click <b>Add Extension</b> and select the folder you just extracted.</li>
</ol>

  <p><b>Video Adblock Delay:</b> There is a known bug where videos may take 3–5 seconds to start playing. I have separated the extensions so the music-only adblock remains unaffected. Feel free to edit the code if you would like to help optimize this!</p>
